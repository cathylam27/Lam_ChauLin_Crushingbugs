# The Crushing Bugs of the Drag and Drop Puzzle game
This is a homework for the 1055 - Multimedia Authoring Class.

# What we need to do?
We should find a solution for two bugs in the Drag and Drop Puzzle game.

The solution of the first bug:
We need to make them don't over one puzzle piece in the one drop zone at the same time.

The solution of the second bug:
We need to create a method for removing or representing back to ready a clear drop zone for players playing the game again.
